FinancialAmount
Namespace: com.matera.dtw.event.common.financial

Monetary amount

Fields

Field Name Type Description
currency com.matera.dtw.event.common.financial.Currency No documentation
amount {'type': 'bytes', 'logicalType': 'decimal', 'precision': 19} Amount as an unscaled value. For values with decimal parts, multiply by the scale. E.g.: $14.85 becomes 1485. $12.00 becomes 1200. The scale is defined according the currency.

FailureInfo
Namespace: com.matera.dtw.command.base.common

Represent a command execution failure metadata. Loosely based on the RFC7807 (https://datatracker.ietf.org/doc/html/rfc7807)

Fields

Field Name Type Description
type string A URI reference RFC3986 (https://datatracker.ietf.org/doc/html/rfc3986) that identifies the problem type.
title string A short, human-readable summary of the problem type.
details ['null', 'string'] A human-readable explanation specific to this occurrence of the problem.
parameters ['null', {'type': 'string', 'logicalType': 'json'}] Additional parameters that give more context on the cause of the given errors. Stored on the Json Format
