Registry APIs

Manage system registries and configuration data.

Endpoints

GET /accounts ▼

Retrieve account information for one or a list of accounts by specifying account keys.

Parameters

key
array
List of account keys (BranchNumber/AccountNumber format)
page
integer
Page number for pagination
size
integer
Number of items per page

Response

{
  "accounts": [
    {
      "accountKey": "1234/567890",
      "branch": "1234",
      "account": "567890",
      "status": "ACTIVE",
      "accountType": "CHECKING",
      "currency": "USD",
      "createdAt": "2024-01-01T10:00:00Z"
    }
  ],
  "pagination": {
    "page": 0,
    "size": 20,
    "totalElements": 1,
    "totalPages": 1
  }
}
GET /memberships/accounts ▼

Retrieve account membership information.

Parameters

page
integer
Page number for pagination
size
integer
Number of items per page

Response

{
  "memberships": [
    {
      "membershipId": "string",
      "accountKey": "1234/567890",
      "groupType": "CORPORATE",
      "status": "ACTIVE",
      "joinedAt": "2024-01-01T10:00:00Z"
    }
  ]
}
GET /accounts/{branch}/{account}/limits ▼

Retrieve account limits information.

Parameters

branch
string
Account branch number
account
string
Account number

Response

{
  "accountKey": "1234/567890",
  "limits": [
    {
      "limitType": "DAILY_TRANSACTION",
      "value": {
        "amount": 10000,
        "currency": "USD"
      },
      "status": "ACTIVE",
      "effectiveDate": "2024-01-01"
    }
  ]
}

Parameters

page
integer

size

Number of items per page

      "value": {
        "amount": 1000000,
        "currency": "USD"
      },
      "description": "Maximum daily transaction limit",
      "status": "ACTIVE"
    }
  ]
}
