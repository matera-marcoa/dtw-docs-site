Digital Twin Balance Services API

This endpoint retrieves information regarding balances from an account or group of accounts.

Base URL

http://localhost:8181/v1

GET /balances

List balances for a group of accounts

Important: This operation requires a logged user with the view-balances role

Parameters

Name Type In Required Description
accountKeys array[string] query Yes A list of account keys, each composed by BranchNumber and AccountNumber separated by a slash (/). Example: 1234/12345,1234/678901
initialDate date query No The initial date for balance retrieval. Format: YYYY-MM-DD. If not specified, returns current balance.
finalDate date query No The final date for balance retrieval. Format: YYYY-MM-DD. Mandatory when initialDate is provided.
balanceTypes array query No Balance types to return (e.g., AVAILABLE, BLOCKED, HELD, BOOK). Returns all types by default.
currency string query No Currency code as per ISO 4217 standard (e.g., USD). Defaults to system currency if not specified.
page integer query No Page number for pagination (min: 0)
size integer query No Maximum number of items per page (1-100)

Responses

200 OK - Successfully retrieved balances
400 Bad Request - Invalid parameters
404 Not Found - Account not found
500 Internal Server Error - Unexpected error

GET /accounts/{branch}/{account}/balances

Get balances list grouped by date for a specific account

Important: This operation requires a logged user with the view-account-balances role

Path Parameters

Name Type Required Description
branch string Yes The branch number
account string Yes The account number
